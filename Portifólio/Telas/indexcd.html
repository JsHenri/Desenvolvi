Amostragem

Amostragem é o processo de selecionar um subconjunto representativo de dados de uma população maior para análise. Isso é feito para economizar tempo e recursos, enquanto ainda permite extrair conclusões precisas e significativas sobre toda a população. A amostragem é fundamental para garantir que as análises sejam realizadas de maneira eficiente e precisa, especialmente quando lidamos com conjuntos de dados muito grandes.

Amostragem aleatória simples: Selecionar aleatoriamente um subconjunto de indivíduos da população. Por exemplo, escolher aleatoriamente 100 clientes de uma lista de 1000 para realizar uma pesquisa de satisfação.

Amostragem Estratificada: Dividir a população em grupos homogêneos (estratos) e, em seguida, selecionar aleatoriamente uma amostra de cada estrato. Por exemplo, dividir os alunos de uma escola por série e, em seguida, selecionar aleatoriamente alguns alunos de cada série para uma pesquisa.

Amostragem Sistematica: Selecionar indivíduos em intervalos regulares de uma lista ordenada da população. Por exemplo, selecionar a cada 10º cliente de uma lista de clientes.

Tendencias Centrais

As tendências centrais são utilizadas para representar um conjunto de dados com um único valor. No estudo da Estatística, é bastante comum que elas sejam utilizadas para compreender melhor o comportamento de um conjunto de dados.

Moda: O valor que ocorre com mais frequência em um conjunto de dados.

Exemplo: No conjunto de dados, a moda é 37 porque ocorre mais frequentemente.

Bimodal: Refere-se a uma distribuição ou conjunto de dados que possui dois picos ou modos.

Exemplo: No conjunto de dados, o bimodal é 1 porque ocorre mais frequentemente.

Polimodal: Refere-se a uma distribuição ou conjunto de dados que possui mais de dois picos ou modos.

Exemplo: No conjunto de dados, o polimodal é 1,3,4,5 porque ocorrem com mais frequencia.

Amodal: Refere-se a uma distribuição ou conjunto de dados que não possui picos claros ou modos distintos.

Exemplo: No conjunto de dados, não existe numeros repetidos, por isso é amodal.

Mediana: O valor do meio em um conjunto de dados ordenados.

Exemplo: No conjunto de dados {3, 6, 7, 8, 10}, a mediana é 7, pois é o valor do meio quando os dados estão em ordem crescente.

Média Aritimetica: A média simples de todos os valores em um conjunto de dados, calculada pela soma de todos os valores dividida pelo número total de valores.

Exemplo: No conjunto de dados {10, 15, 20, 25}, a média aritmética é (10 + 15 + 20 + 25) / 4 = 17.5.

Média Aritimetica Ponderada: Uma média onde cada valor tem um peso associado a ele. A média é calculada multiplicando cada valor pelo seu peso, somando-os e dividindo pela soma dos pesos.

Exemplo: Se as notas de um aluno em um curso têm pesos diferentes, como 40% para trabalhos, 30% para provas e 30% para participação, a média ponderada seria calculada multiplicando cada nota pela sua ponderação, somando-as e dividindo por 100.

Percentil: Um valor abaixo do qual uma determinada porcentagem de observações em um grupo de observações cai.

Exemplo: Nesse exemplo exibe o percentil de 40% que é igual a 54 e o percentil de 50% que é igual a 64.

Quartil: Um ponto que divide um conjunto de dados em quatro partes iguais, onde cada parte representa 25% dos dados.

Exemplo: Nesse exemplo exibe o quartil de 25% que é igual a 38,5, o quartil de 50% que é igual a 64 e o quartil de 75% que é igual a 93%

Meididas de Disperção

Medidas de dispersão são estatísticas que indicam os quão espalhados ou variáveis são os dados em um conjunto de observações. Elas ajudam a entender a dispersão dos dados em torno da média e fornecem insights sobre a consistência dos dados. A amplitude é uma medida de dispersão simples que indica a diferença entre o maior e o menor valor em um conjunto de dados.

Variância: A média dos quadrados das diferenças entre cada valor e a média do conjunto de dados. Quanto maior a variância, maior a dispersão dos dados.

Exemplo: Considere os dados {2, 4, 4, 4, 5, 5, 7}. A média é 4. A variância seria calculada como ((2-4)² + (4-4)² + (4-4)² + (4-4)² + (5-4)² + (5-4)² + (7-4)²) / 7 = 2.

Desvio Padrão: A raiz quadrada da variância. Ele fornece uma medida de dispersão em termos da unidade de medida original.

Exemplo: No conjunto de dados acima, o desvio padrão seria a raiz quadrada de 2, que é aproximadamente 1.41.

Limites em Ciência de Dados

Limites geralmente se referem a intervalos específicos de valores que são usados para definir critérios, restrições ou filtros em uma análise. Esses limites podem ser aplicados em várias etapas do processo de análise de dados para segmentar ou filtrar os dados de acordo com as necessidades específicas do problema em questão.

Limite de valor minimo e máximo: Definir limites para os valores de uma variável. Por exemplo, ao analisar dados de temperatura, podemos definir um limite mínimo e máximo para considerar apenas as leituras dentro de um determinado intervalo, como entre 0°C e 40°C.

Limite de percentil: Usar percentis para definir limites com base na distribuição dos dados. Por exemplo, ao avaliar a renda de uma população, podemos definir um limite superior de 90º percentil para identificar os 10% mais altos da distribuição de renda.

Limite de Desvio Padrão: Definir limites com base no desvio padrão dos dados. Por exemplo, ao analisar o desempenho de um modelo preditivo, podemos definir limites para identificar pontos de dados que estão além de um certo número de desvios padrão da média.

Integrais

Integrais em ciência de dados referem-se a conceitos matemáticos fundamentais usados em análises estatísticas e computacionais para calcular áreas, volumes, distâncias e outras grandezas sob curvas ou superfícies.

Definição: Uma integral é uma medida de acumulação contínua. Em termos simples, ela calcula a soma de infinitos incrementos infinitesimais de uma função ao longo de um intervalo específico.

Primitivas: As primitivas, também conhecidas como antiderivadas, são o oposto das derivadas. Se a derivada de uma função é conhecida, a primitiva é a função original a partir da qual a derivada foi calculada.

Distancia: Em ciência de dados, integrais são usadas para calcular distâncias entre pontos em um espaço multidimensional. Por exemplo, a integral de uma função de densidade de probabilidade pode ser usada para calcular a distância entre duas distribuições de probabilidade.

Area: Uma das aplicações mais comuns das integrais é o cálculo de áreas sob curvas. Isso é útil, por exemplo, para determinar a probabilidade de eventos em distribuições de probabilidade contínuas.

Volume: Integrais também são usadas para calcular volumes de sólidos tridimensionais. Por exemplo, ao analisar conjuntos de dados volumétricos em imagens médicas ou modelos 3D, integrais são frequentemente usadas para calcular volumes de regiões de interesse.

Integral Definida: Uma integral definida é uma integral que possui limites de integração específicos, indicando os pontos inicial e final do intervalo sobre o qual a integral é calculada. A integral definida resulta em um valor numérico que representa a área ou o volume acumulado sob a curva ou superfície no intervalo especificado.

Raizes Racionais

As raízes racionais são os possíveis valores racionais (ou seja, números que podem ser expressos como uma fração de dois inteiros) de uma equação polinomial. Elas são importantes porque fornecem pistas sobre possíveis soluções para a equação.
